# Infrastruktur

Dokumentation meines Homelab-Clusters. Enthält aktuelle Architektur, Netzwerkplanung, Iterationshistorie sowie weitere Dokumente.
*Hostnamen sowie User sind in der kompletten Dokumentation anonymisiert*

## Dokumente

| Datei                     | Inhalt                                          |
|---------------------------|-------------------------------------------------|
| `architecture.md`         | Architektur der gesamten Infrastruktur          |
| `network-plan.md`         | IP-Planung, WireGuard Mesh                      |
| `deployment-plan.md`      | Aufbauphasen und offene Baustellen              |
| `iterations-history.md`   | Lernweg mit gescheiterten Versuchen             |
| `CHANGELOG.md`            | Changelog zum Cluster                           |
| `baseline_checkliste.md`  | Aktuellen Baseline Setup Prozedere              |
| `ssh.md`                  | Runbook zum SSH Setup                           |

Die Iterationshistorie dokumentiert auch verworfene Ansätze, um Designentscheidungen nachvollziehbarer zu machen.
