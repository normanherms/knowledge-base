# Persönliche Knowledge-Base

## Einführung

Dies ist meine persönliche Dokumentation zu allen Lerninhalten aus Udemy, Büchern, Kursen und praktischer Erfahrung.
Bisherige Inhalte wurden händisch aufgezeichnet und anschließend digitalisiert. Keine dieser Inhalte wurde aus Kursen Wort für Wort transkribiert.

---

## Ziele

- Stetiges Lernen dokumentieren
- Eine überall verfügbare, reduzierte Dokumentation auf das Wesentliche
- Bessere Beherrschung von Markdown

Dieses Repository ist public und kann von jedem genutzt werden, dem es hilft.
